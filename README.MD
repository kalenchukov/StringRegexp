# StringRegexp

Реализации регулярных выражений для строк.

## Использование

#### Заголовок

```java

/*
Результат выполнения:
{

}
*/
```

## Установка

Для установки этого проекта, введите в терминале:

```bash
git clone https://github.com/kalenchukov/StringRegexp.git
```
